### 加密与解密基础

#### 核心分类
1. **对称加密方式**:
   - 使用相同密钥进行加密和解密。
   - 代表算法：AES (Advanced Encryption Standard)。
   - 特点：处理速度快，适合大量数据。

2. **非对称加密方式**:
   - 使用一对密钥：公钥和私钥。
   - 代表算法：RSA (Rivest-Shamir-Adleman)。
   - 特点：不需要交换密钥，安全性高但速度慢。

#### RSA算法详解
**密钥生成步骤**:
1. 选择两个大素数 `p` 和 `q`。
2. 计算 `n = p * q`。
3. 计算欧拉函数 `φ(n) = (p-1)(q-1)`。
4. 选择一个整数 `e`，满足 `1 < e < φ(n)` 且 `e` 与 `φ(n)` 互质。
5. 计算 `d` 使得 `d * e ≡ 1 mod φ(n)`。

**加密与解密**:
- 加密: `C = M^e mod n`（M为明文，C为密文）
- 解密: `M = C^d mod n`

**示例（Python）**:
```python
def rsa_encrypt(M: int, e: int, n: int) -> int:
    return pow(M, e, n)  # M^e % n

def rsa_decrypt(C: int, d: int, n: int) -> int:
    return pow(C, d, n)

# 示例参数
p, q = 61, 53  # 素数
n = p * q       # 3233
phi = (p-1)*(q-1)  # 3120
e = 17          # 公钥
d = pow(e, -1, phi)  # 私钥，2753

M = 123
C = rsa_encrypt(M, e, n)  # 加密
M_dec = rsa_decrypt(C, d, n)  # 解密
print(f"明文: {M} -> 密文: {C} -> 解密: {M_dec}")
```

**例题**:
1. 公钥 `(e, n) = (3, 33)`，明文M=5，求密文C。
   - 解答: `C = 5^3 mod 33 = 125 mod 33 = 26`。

2. 已知 `p=3, q=11, e=7`，求私钥d。
   - 解答: 
        `n = 33, φ(n) = (3-1)(11-1)=20`
        `7 * d ≡ 1 mod 20` -> `d=3` (因为7*3=21≡1 mod 20)

#### 实际应用
**SSL/TLS协议中的密钥交换**:
1. 客户端请求服务器的公钥。
2. 服务器发送公钥。
3. 客户端生成对称密钥，用服务器的公钥加密后发送给服务器。
4. 服务器用私钥解密得到对称密钥。
5. 后续通信使用对称密钥加密。

#### 学习要点
1. 对称加密的密钥长度：AES通常为128/192/256位。
2. 数字签名：使用私钥签名，公钥验证。
3. 混合加密系统：使用非对称加密交换对称密钥，再用对称加密加密数据。
